# How to Install SEOQaz Chrome Extension

## Installation Steps

### Step 1: Download the Extension
Clone this repository or download it as a ZIP file:
```bash
git clone https://github.com/muhamed-kanapiya/seoqaz-extension.git
```

Or download the ZIP and extract it to a folder.

### Step 2: Open Chrome Extensions Page
1. Open Google Chrome browser
2. Navigate to `chrome://extensions/`

### Step 3: Enable Developer Mode
In the top-right corner of the Extensions page, toggle the **Developer mode** switch to ON.

### Step 4: Load the Extension
1. Click the **"Load unpacked"** button that appears after enabling Developer mode
2. Browse to the `seoqaz-extension` folder
3. Click **"Select Folder"**

### Step 5: Verify Installation
- The SEOQaz extension icon should now appear in your Chrome toolbar

## How to Use

1. **Navigate to any webpage** you want to analyze
2. **Click the SEOQaz icon** in your Chrome toolbar
3. **Click "Analyze Page"** button in the popup
4. **Review the results** including:
   - Title tag analysis
   - Meta description check
   - Heading structure
   - Image alt text coverage
   - Link analysis
   - Open Graph tags
   - Overall SEO score

## Troubleshooting

### Extension not showing up?
- Make sure Developer mode is enabled
- Try refreshing the Extensions page
- Check that you selected the correct folder

### "Analyze Page" not working?
- Make sure you're on a valid webpage (not chrome://, edge://, or other browser pages)
- Try refreshing the webpage and clicking the extension again
- Check the browser console (F12) for any errors

### Need to update the extension?
1. Make changes to the files
2. Go to `chrome://extensions/`
3. Click the refresh icon (ðŸ”„) on the SEOQaz extension card

## Uninstallation

1. Go to `chrome://extensions/`
2. Find the SEOQaz extension
3. Click **"Remove"**
4. Confirm the removal
